Surgical Scheduling Dashboard

How might we create a user-friendly dashboard for decision-makers in the surgical department?

Role: Product Designer Duration: July 2021 - Present Team: Methodology (2), Development (2), Product (me)

The Challenge

"There is a lack of communication and automation in the process of surgical scheduling among the main stakeholders" in the surgical department. The team wanted to create a graphical user interface for surgical scheduling planning tool for surgical team for one of the largest hospitals in North America.

Project Goals

Decision-Making Information

Provide decision-making information for hospital administrators and leadership.

Easier Access

Enable easier access to critical information for all stakeholders.

User-Friendly Interface

Create a user-friendly interface for users with varying data experience levels.

My Design Responsibilities

User Research & Ideation

Conducted user research and solution ideation to understand scheduling challenges and stakeholder needs.

High-Fidelity Prototyping

Developed high-fidelity Figma prototype with accessibility guidelines and user-centered design principles.

Web Implementation

Created web page for Orchid Analytics to showcase the dashboard solution.

Key Deliverables

User Research Materials

User interview questions and surveys designed to understand stakeholder needs and pain points.

User Flow Design

User flows for both direct and indirect surgical planning stakeholders to map the entire process.

Design System

Basic design system with low-fidelity sketches progressing to high-fidelity prototypes.

Accessibility Focus

High-fidelity Figma prototype with accessibility guidelines for diverse user experience levels.

Project Scope

The dashboard was focused on two key medical services: Urology and Orthopedics, allowing for targeted design solutions for these specific surgical departments.

Note: Due to the confidential nature of this healthcare project, only an overview of the design process can be shared publicly.

Project Impact

The project resulted in a comprehensive design solution for surgical scheduling, addressing critical communication and automation challenges in one of North America's largest hospitals.
